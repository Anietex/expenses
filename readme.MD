# Expense Manager


## How to install
After cloning the repo  copy .env.example to .env, update your database configs then run

`composer install`  
`npm install`  
`php artisan key:generate`  
`php artisan migrate`  
`php artisan serve`

## Tech Stack
[Laravel](https://laravel.com)  
[VueJs](https://vuejs.org)  
[Vue Router](https://router.vuejs.org)
